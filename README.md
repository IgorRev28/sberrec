# Предсказание следующего товара в сессии

По истории покупок пользователя предсказываю, какой товар он купит следующим.

## Как работает модель

Строю граф переходов между товарами — смотрю, после какого товара какой чаще всего покупают. Для предсказания беру последний товар из сессии и смотрю, куда чаще всего переходят с него. Если рекомендаций меньше 10 — добиваю самыми популярными товарами.

## Результаты

Базовая модель (топ популярных) дала 38.4%. Моя модель (граф переходов) — 51.4%. Улучшение — 13%.

## Метрика

Hit@10 — попадает ли правильный товар в топ‑10 рекомендаций. 51% значит, что в половине случаев модель угадала.

## Структура проекта

`src/` — код, `images/` — графики (длина сессий, популярность товаров, сравнение моделей), `sessions.jsonl` — данные.
